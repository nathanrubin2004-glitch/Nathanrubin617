  <action>
Update styles.css to replace the display:none/flex toggle approach with a CSS animation that slides the panel up on open and down on close.

The current implementation uses:
```css
.chat-window {
    display: none;
    ...
}
.chat-window.active {
    display: flex;
}
```

This produces a jarring instant show/hide. Replace with an animation approach.

**Implementation approach:**

Since CSS cannot animate `display` property directly, use a two-stage approach:
- Keep `display: flex` always, but use `visibility`, `opacity`, and `transform` to show/hide
- Use a short transition for smooth animation

Replace the `.chat-window` and `.chat-window.active` rules with:

```css
.chat-window {
    position: fixed;
    bottom: 8rem;
    right: 2rem;
    z-index: 40;
    width: 100%;
    max-width: 400px;
    height: 550px;
    background: white;
    border-radius: 12px;
    box-shadow: 0 5px 40px rgba(0, 0, 0, 0.16);
    display: flex;
    flex-direction: column;
    overflow: hidden;
    /* Animation: hidden by default */
    opacity: 0;
    visibility: hidden;
    transform: translateY(16px);
    transition: opacity 0.25s ease, transform 0.25s ease, visibility 0s linear 0.25s;
    pointer-events: none;
}

.chat-window.active {
    opacity: 1;
    visibility: visible;
    transform: translateY(0);
    transition: opacity 0.25s ease, transform 0.25s ease, visibility 0s linear 0s;
    pointer-events: auto;
}
```

Key points:
- `display: flex` stays on always — enables the flexbox layout without toggling display
- `opacity: 0` + `visibility: hidden` hides the element (visibility:hidden prevents interaction, opacity:0 enables transition)
- `transform: translateY(16px)` provides the slide-up effect on open
- `transition` on `.chat-window` has `visibility 0s linear 0.25s` (delayed, so visibility hides AFTER opacity fades out)
- `transition` on `.chat-window.active` has `visibility 0s linear 0s` (immediate, so visible RIGHT AWAY on open)
- `pointer-events: none` when hidden prevents accidentally clicking through the invisible panel
