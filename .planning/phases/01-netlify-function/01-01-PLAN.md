```json

  "type": "module"
}
```
No dependencies, no scripts, no name field. This does NOT require `npm install` (per D-01/D-02 no-npm decision).

2. `netlify.toml` — Explicit Netlify Functions configuration. Must contain:
```toml
[functions]
  directory = "netlify/functions"
```
This tells Netlify where to find serverless functions. While Netlify auto-detects `netlify/functions/`, explicit config avoids surprises.

Do NOT add any other fields to either file. No `node_bundler`, no `[build]` section, no dependencies.

Create `netlify/functions/chat.js` as a single ESM file implementing the chat API proxy. The function must follow the modern Netlify Functions format: `export default async function(req)` returning `new Response(...)`.

**Structure (in order):**

1. **SYSTEM_PROMPT constant** (per D-04, D-07, D-08, D-09) — A const string at the top of the file containing Nathan's bio derived from the HTML pages. Use this exact content:

```
You are an AI assistant on Nathan Rubin's personal portfolio website (nathanrubin617.com). Your role is to help visitors learn about Nathan and his work.

ABOUT NATHAN:
Nathan Rubin is a writer, athlete, economist, and storyteller from Boston, MA. He studies Economics, Community Relations, and Diaspora at the Liberal Arts College of UMass Boston. He is passionate about inspiring the next generation through storytelling and community leadership.

HIS BOOK — "CHASING A DREAM":
Nathan's debut children's book teaches confidence, kindness, and self-belief, encouraging young readers to see the greatness within themselves. Available at local libraries, community centers, youth organizations, and Little Free Library locations. Purchase: https://buy.stripe.com/5kQ8wR1pRfD2dxV1wwgYU02

BASKETBALL:
Nathan is a 2024 graduate of Lexington Christian Academy (EIL Champion, NEPSAC Champion). He now plays basketball at UMass Boston.

CONTACT:
Email: nathanrubin2004@gmail.com | Phone: (617) 840-0068

BEHAVIOR:
Answer warmly about Nathan, his book, his basketball career, and how to contact him. If asked about unrelated topics, briefly acknowledge and redirect back to Nathan and what you can help with on this site. Keep responses concise and conversational.
```

2. **corsHeaders() helper function** — Returns an object with:
   - `'Access-Control-Allow-Origin': '*'`
   - `'Access-Control-Allow-Methods': 'POST, OPTIONS'`
   - `'Access-Control-Allow-Headers': 'Content-Type'`

3. **Default export async function(req)** — The handler, implementing this flow:

   a. **OPTIONS check** (per D-06, FUNC-06): If `req.method === 'OPTIONS'`, return `new Response(null, { status: 200, headers: corsHeaders() })`.

   b. **Method guard**: If `req.method !== 'POST'`, return HTTP 405 with `{ error: 'Method not allowed' }` and `Content-Type: application/json` header.

   c. **API key check** (per D-03, FUNC-03): Read `process.env.ANTHROPIC_API_KEY`. If falsy, return HTTP 500 with `{ error: 'API key not configured' }` and both Content-Type and CORS headers.

   d. **Parse request body**: Wrap `await req.json()` in try/catch. Extract `messages` from the body. If parsing fails, return HTTP 400 with `{ error: 'Invalid request body' }`. If `messages` is not an array or is empty, return HTTP 400 with `{ error: 'messages array is required' }`. Both with Content-Type and CORS headers.

   e. **Call Anthropic API** (per D-01, D-10, D-11, FUNC-02, FUNC-07): Wrap the fetch in try/catch.
   ```javascript
   fetch('https://api.anthropic.com/v1/messages', {
     method: 'POST',
     headers: {
       'x-api-key': apiKey,
       'anthropic-version': '2023-06-01',
       'content-type': 'application/json'
     },
     body: JSON.stringify({
       model: 'claude-sonnet-4-20250514',
       max_tokens: 500,
       system: SYSTEM_PROMPT,
       messages
     })
   })
   ```
   If fetch throws (network error), return HTTP 500 with `{ error: 'Failed to reach Anthropic API' }`.

   f. **Check Anthropic response**: If `!anthropicResponse.ok`, return HTTP 500 with `{ error: 'Anthropic API error' }` and CORS headers.

   g. **Extract reply** (per D-04): Parse response JSON, extract `data.content[0].text`, return HTTP 200 with `{ reply }` and both Content-Type and CORS headers.

**Critical anti-patterns to avoid:**
- Do NOT use `module.exports.handler` (legacy CommonJS — per D-03)
- Do NOT use `Netlify.env.get(...)` (Edge Functions only — use `process.env`)
- Do NOT use `req.body` directly (Web Platform Request requires `await req.json()`)
- Do NOT return `{ statusCode, body }` objects (must use `new Response(...)`)
- Do NOT import `@anthropic-ai/sdk` or any npm package (per D-01/D-02)
- Do NOT accept a system prompt from the client request (per D-09)
